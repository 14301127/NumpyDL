# Deep Learning Framework based on Numpy

## requirements
 
- Python3
- Numpy
- Matplotlib
- Scipy
- xlwt
- scikit-learn

## design philosophy
In every task, **_the operation flow/process is unchanged_** and **_the only changing 
thing is just the specific operation in each process_**. (**业务流程是不变的，变化的
只是具体的业务。**)
