
# Chatbot


## Usage

Step 1: Decompress ``data.tar.gz`` into the ``data`` directory.

    cd applications/chatbot/data
    tar -xzf data.tar.gz

Step 2: Run ``main.py`` script.
    
    python main.py


Step 3: Open your favourite browser, and type ``http://127.0.0.1:5000/``

    Enjoy!
